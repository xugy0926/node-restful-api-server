以下 api 路径均以 `https://js.xinshengdaxue.com` 为前缀


## 获取心里话

```
[GET] /api/v1/learnJS/course/:id/words

id: 第一期课程为1
```

#### 结果
```
// 处理正常code:1
{
  code, // 1: 处理正常，0：处理错误
  message, // code === 0 时返回
  words
}
// 处理错误code:0
{
  code, 
  message
}
```

## 对老师有话说

```
[POST] /api/v1/learnJS/sayToMe

body: {
  name: '你的大名',
  account: '新生大学注册时用的手机号或邮箱',
  content: '对老师说的话'
}
```

#### 结果
```
// 处理正常code:1
{
  code, // 1: 处理正常，0：处理错误
  message, // code === 0 时返回
  data,
}
// 处理错误code:0，表单不完整
{
  code, 
  message: name or account or content is null
}
// 处理错误code:2，新大账号不匹配
{
  code, 
  data
}
```
